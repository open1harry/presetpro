# Preset Pro

Preset Pro is a Unity Editor productivity plugin for fast hierarchy presets.

## Features

- Folder-based categories and prefab-based items
- `Tools -> Preset Pro -> Settings` configuration window
- `Shift + V` quick add flow from selected Hierarchy object
- Auto-generated native menu entries under `GameObject/Preset Pro/...`
- First-run introduction window
- Chinese UI by default, with optional English switch

## Unity Version

- Developed for Unity `2022.3.x`

## Package Structure

- `package.json`
- `Editor/`

## Basic Usage

1. Open `Tools -> Preset Pro -> Settings`.
2. Confirm or change presets root folder.
3. Put prefabs under category folders.
4. Click `Generate/Refresh Menu`.
5. Use `Shift + V` to save selected scene object as a preset.
